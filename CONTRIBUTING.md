# Contributing

Compost is a personal project shared as-is. It is designed to be a 10+ year single-user tool — **your own personal AI brain**, cloned from this template and grown locally with your own knowledge.

## This repo does not accept pull requests

If you want to change how Compost works, **fork it**. Your fork is yours to modify however you want.

We don't merge external changes because:

- Compost is highly personal — the author's choices reflect their workflow, not a shared product roadmap
- Maintaining a shared implementation creates community management overhead that conflicts with the "personal tool" philosophy
- Your fork can diverge as much as you want without coordinating with anyone

## Issues are used for personal bug tracking

Feel free to open issues to:

- Report bugs you hit while using Compost
- Share ideas or questions

We don't guarantee responses. Issues are primarily a scratchpad for the author. If your fork diverges significantly, maintain your own issue tracker there.

## How to use Compost for yourself

1. Fork or clone this repo
2. Follow the Quick Start in [README.md](README.md)
3. Grow your own knowledge base locally
4. Modify the code freely — it's yours once cloned

Your data never leaves your machine (see [SECURITY.md](SECURITY.md)).
